GET: Search for Student by Last Name

Use this endpoint to search for students by their last name.

Request:

        
HTTP Method: GET
Endpoint: /students?last_name=Smith

Response:

        
Status Code 200 - OK
Response Body:
[
    {
        "record_id": 2,
        "first_name": "Jane",
        "last_name": "Smith",
        "gpa": 3.9,
        "enrolled": false
    }
    // More student objects with the last name "Smith"...
]
