GET: Display a Single Student by ID

Use this endpoint to display a single student by their ID.

Request:

        
HTTP Method: GET
Endpoint: /students/{student_id}

Response:

        
Status Code 200 - OK
Response Body:
{
    "record_id": {student_id},
    "first_name": "John",
    "last_name": "Doe",
    "gpa": 3.8,
    "enrolled": true
}
