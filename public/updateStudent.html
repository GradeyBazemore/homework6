PUT: Update a Student by ID

Use this endpoint to update a student by their ID.

Request:

        
HTTP Method: PUT
Endpoint: /students/{student_id}
Content-Type: application/json
Request Body:
{
    "first_name": "Updated First Name",
    "last_name": "Updated Last Name",
    "gpa": 3.9,
    "enrolled": false
}

Response:

        
Status Code 200 - OK
Response Body:
{
    "message": "Student with ID {student_id} updated successfully."
}
