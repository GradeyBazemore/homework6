Create a New Student

Use this endpoint to create a new student.

Request:

        
HTTP Method: POST
Endpoint: /students
Content-Type: application/json
Request Body:
{
    "first_name": "Chris",
    "last_name": "Bazemore",
    "gpa": 3.8,
    "enrolled": true
}

Response:

        
Status Code 201 - Created
Response Body:
{
    "record_id": 123,
    "first_name": "Gwen",
    "last_name": "Jack",
    "gpa": 3.8,
    "enrolled": true
}
